Fullstack AI Journey

7 Weeks of Building, Learning & Sharing

🚀 What is This?

A personal 7-week challenge to build fullstack web apps with AI integration, share progress daily, and grow in public.

🛠 Tech Stack

  • Frontend: Next.js (React)
  • Backend: Django (Python)
  • AI/ML: NumPy, Pandas, Scikit-learn, Matplotlib
  • Visualization: Charts & Graphs on the frontend

📅 Challenge Structure

Duration: 7 Weeks
Posting: 6 Days/Week
Daily Includes:

  • What I learned
  • Code written
  • Features built
  • ML models & data
  • Visual outputs (graphs/screenshots)

📢 Where to Follow

  • LinkedIn: Daily posts
  • GitHub: Repos per feature/day
  • Video: Screen recordings with custom thumbnails

🎯 Goals

  • Master fullstack web dev + AI/ML integration
  • Serve ML models through real frontend apps
  • Build in public and stay consistent
  • Inspire others to join or create similar journeys
